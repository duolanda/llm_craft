# 想法
- 切屏速度同样也是区分新手老手的一环，人类在打 rts 的时候也是主动选择去看哪里的。所以相比每轮给 AI 发大量 status，改成 AI 主动去看一些信息也不错
- 拿人机当 baseline
  - 一种是几乎随机决策的电脑，一种是先生产几个农民采集然后无脑爆兵冲基地的电脑
  - 如果 llm 和后者对打胜率不能超过 80%，说明我们设计还有问题，如果 llm 和随机打胜率都不能超过 50%，那说明问题大了
- 现在确实只是不停发即时消息，应该有更宏观和持久的战略，但同时又要有精彩的临场反应（分层处理？）
- 增加强制攻击，如果 AI 打友军会很有节目效果
- worker 可以修建筑
- 迷雾是 RTS 的底线
- 等到了后期，可以搞赛季，避免有人找到了万能解法，比如
  - worker 采集量减半，鼓励前期 rush
  - Braracks 造假减半，鼓励爆兵
  - HQ 血量减半，偷家收益更大等
- 等 RTS 成熟之后，搞一个类似 War3 地图编辑器的东西，将会是绝杀，LLM 可以玩几乎任何游戏
